The following PVC will ask for Storage to the existing PV.

```YAML
kind: PersistentVolumeClaim
apiVersion: v1
metadata:
  name: pv-claim
spec:
  storageClassName: demo
  accessModes:
    - ReadWriteOnce
  resources:
    requests:
      storage: 1Gi
```

```bash
kubectl apply -f 00-pvc.yaml
```

```bash
kubectl get pvc
```
```bash
kbuectl describe pvc pv-claim
```
```bash
kubectl get pvc,pv
```
